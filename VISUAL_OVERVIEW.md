## 📡 API Call Structure

```
POST /api/GraduateAuth/register
POST /api/StudentAuth/register
POST /api/trainingcenterauth/register
POST /api/UniversityAuth/register

Headers:
├── Content-Type: multipart/form-data
└── Authorization: Bearer {token} (if needed)

Request Body:
├── email
├── password
├── firstName
├── lastName
├── country
├── city
├── [education fields]
├── [experience fields]
├── [optional fields]
└── [file uploads]

Response:
├── Success (200-299)
│  ├── token
│  ├── user_id
│  └── message
└── Error (400-599)
   ├── error_code
   ├── message
   └── details
```
